### 关于var关键字需要知道的
- 对于var关键字，要么定义全局变量、要么定义函数内变量；
- 不能定义只在一个语句块儿内部起作用的变量；
- 这叫做js的函数级作用域(function-level scope)；

```
(function test(){
    if (true) {
      x = 5;  //x的作用域为全局
    }
    console.log(x); // 5
})();
```

```
function foo() { 
 	var x = 1; 
 	if (x) { 
 		(function () { 
 			var x = 2; 
 		}()); 
 	} 
 	console.log(x);  //1
 } 
foo();
```
